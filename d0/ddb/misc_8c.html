Detailed Description

Some random but useful functions.

Optional #defines

  • USE_CPU_SPEED - enables getCpuSpeed(), and setCpuSpeed()
  • USE_GET_TIME - enables getTime()
  • USE_WAIT8 - enables wait()
  • USE_ISQRT - enables isqrt()
